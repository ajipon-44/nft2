## プロジェクトについて

ブロックチェーンベースの銀行アプリ

<!-- プロジェクトの概要を記載 -->

NFT を所持している人だけが，送金，預入，引き出し機能を使用できる．<br>
NFT を所持していない人はお金の受け取りは可能だが，他の処理はできない．
